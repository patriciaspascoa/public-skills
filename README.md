# Public Skills — Patrícia Sales

Skills para Claude Code de uso geral. Gratuitas e abertas.

## Instalação

```bash
claude plugin install github:patriciaspascoa/public-skills
```

## Skills disponíveis

| Skill | Descrição |
|---|---|
| humanizer | Elimina AI Slop e reescreve textos para soar natural, preservando a voz da autora |

## Como usar

Após instalar, use `/humanizer` no Claude Code e cole o texto que quer humanizar.
